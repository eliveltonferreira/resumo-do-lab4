## Resumo do lab Computação e Rede

Sobre como descrever as opçoes de maquinas virtuais e vms, conjunto de dimensionamento de maquinas virtuais.
Conjunto de disponibilidade e area de trabalho remota.
A Computação do Azure é um serviço sob demanda que fornece recursos de computação, como discos, processadores, memória, rede e sistemas operacionais.
Oferta de IaaS que oferece personalização e controle total.
Conjuntos de dimensionamento 
Escalar horizontalmente quando o recurso precisar aumentar.
Reduzir horizontalmente quando o recurso precisar diminuir.
A Área de Trabalho Virtual do Azure é uma virtualização de área de trabalho e aplicativo executada na nuvem.
Os contêineres do Azure fornecem um ambiente leve e virtualizado que não exige o gerenciamento do sistema operacional e pode responder a alterações sob demanda. 
Aplicativos de Contêiner do Azure: uma oferta de PaaS, como instâncias de contêineres, que pode balancear a carga e escalar.
Azure Functions
O código baseado em eventos é executado quando chamado, sem exigir uma infraestrutura de servidor durante períodos inativos é uma oforta Paas.
Rede Virtual do Azure (VNet) permite que os recursos do Azure se comuniquem uns com os outros, com a Internet e com as redes locais.
VPN Azure serviço que permite conectar sua rede local (on-premises) à nuvem Azure de forma segura e privada. A VPN Azure cria um túnel virtual criptografado entre os dois ambientes, garantindo que seus dados sejam transmitidos de forma segura.
